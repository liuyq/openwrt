# 从左至右依次是VCC（不接）,GND,TX,RX分别接上
# Putty终端连接，用tpl中断，打开tftp服务，网线接在任意一个lan口，bin包放在tftp目录下
# 命令如下
# tftp 0x80800000 ***.bin
# erase 0x9f020000 +0x3c0000
# cp.b 0x80800000 0x9f020000 0x3c0000
# bootm 0x9f020000
# 自动重启就好了
# 重新刷好了
# TP1->RX <-> ttl green
# TP2->TX <-> ttl white
